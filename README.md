# OWASP Juice Shop — QA Project

Учебный проект по ручному тестированию web-приложения OWASP Juice Shop.

## Цель проекта

Закрепить навыки ручного тестирования web-приложений, работы с тестовой документацией, Chrome DevTools, REST API и Postman.

## Объект тестирования

OWASP Juice Shop — учебное web-приложение, имитирующее интернет-магазин.

## Что будет в проекте

- чек-лист;
- тест-кейсы;
- баг-репорты;
- результаты тестирования API;
- Postman-коллекция;
- итоговый отчёт о тестировании.

## Инструменты

- Chrome;
- Chrome DevTools;
- Postman;
- Git и GitHub.
