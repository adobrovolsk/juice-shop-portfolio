# Отчёт о тестировании OWASP Juice Shop

## Environment

- OWASP Juice Shop 20.1.1
- Windows 11
- Google Chrome
- Postman

## Scope

Проведено тестирование следующих разделов:

- регистрация и авторизация;
- каталог и поиск товаров;
- корзина;
- REST API;
- базовые проверки безопасности.

## Results

- В рамках выполненных проверок дефекты в основных пользовательских сценариях не выявлены.
- Выполнены 3 API-проверки — все завершены со статусом `Passed`.
- Выявлено 2 дефекта безопасности.
- Оформлено 2 баг-репорта.
- Подготовлены чек-лист, тест-кейсы и Postman-коллекция.

## Detected defects

1. Обход авторизации с помощью SQL-инъекции.
2. Выполнение JavaScript-кода через XSS в поле поиска.

## Conclusion

В рамках проведённых функциональных и API-проверок дефекты в работе разделов регистрации, авторизации, каталога, поиска и корзины не выявлены.

При проверках безопасности обнаружены две уязвимости: обход авторизации с помощью SQL-инъекции и выполнение JavaScript-кода через XSS в поле поиска.
