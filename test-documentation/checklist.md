# Чек-лист проверок OWASP Juice Shop

## Область тестирования

- Регистрация пользователя
- Авторизация пользователя
- Каталог товаров
- Поиск товаров
- Корзина
- Безопасность

## Регистрация пользователя

| ID | Проверка | Ожидаемый результат | Статус |
|---|---|---|---|
| REG-001 | Регистрация с валидными данными | Пользователь успешно зарегистрирован | Not Run |
| REG-002 | Регистрация с уже зарегистрированным email | Регистрация не выполняется, отображается сообщение об ошибке | Not Run |
| REG-003 | Регистрация с пустыми обязательными полями | Кнопка регистрации неактивна, отображаются сообщения об ошибке  | Not Run |
| REG-004 | Регистрация с email неверного формата | Регистрация не выполняется, отображается сообщение об ошибке | Not Run |
| REG-005 | Регистрация с несовпадающими паролями | Регистрация не выполняется, отображается сообщение об ошибке | Not Run |

## Авторизация пользователя

| ID | Проверка | Ожидаемый результат | Статус |
|---|---|---|--|
| AUTH-001 | Авторизация с валидными данными | Пользователь успешно входит в учётную запись | Not Run |
| AUTH-002 | Авторизация с валидным email и неверным паролем | Вход не выполняется, отображается сообщение об ошибке | Not Run |
| AUTH-003 | Авторизация с несуществующим email и валидным паролем | Вход не выполняется, отображается сообщение об ошибке | Not Run |
| AUTH-004 | Попытка авторизации с пустыми полями | Кнопка «Log in» неактивна, отображаются сообщения об ошибке | Not Run |
| AUTH-005 | Авторизация с email неверного формата и непустым паролем | Вход не выполняется, отображается сообщение об ошибке | Not Run |
| AUTH-006 | Авторизация с валидным email и пустым паролем | Кнопка «Log in» неактивна, отображается сообщение об ошибке | Not Run |

## Каталог товаров

| ID | Проверка | Ожидаемый результат | Статус |
|---|---|---|---|
| CAT-001 | Открытие каталога товаров | Каталог загружается без ошибок, отображается список товаров | Not Run |
| CAT-002 | Отображение данных в карточке товара | В карточке отображаются название, изображение, цена и кнопка добавления в корзину | Not Run |
| CAT-003 | Открытие подробной информации о товаре | Открывается страница с информацией о выбранном товаре | Not Run |
| CAT-004 | Закрытие подробной информации о товаре | Информация о товаре закрывается, пользователь возвращается в каталог | Not Run |

## Поиск товаров

| ID | Проверка | Ожидаемый результат | Статус |
|---|---|---|---|
| SEARCH-001 | Поиск по полному названию существующего товара | Отображается товар, соответствующий поисковому запросу | Not Run |
| SEARCH-002 | Поиск по части названия существующего товара | Отображаются товары, содержащие введённый текст в названии | Not Run |
| SEARCH-003 | Поиск несуществующего товара | Товары не отображаются, программа не выдаёт системную ошибку | Not Run |

## Корзина

| ID | Проверка | Ожидаемый результат | Статус |
|---|---|---|---|
| CART-001 | Добавление одного товара в корзину | Товар добавляется в корзину, счётчик корзины обновляется | Not Run |
| CART-002 | Повторное добавление того же товара | Количество товара в корзине увеличивается | Not Run |
| CART-003 | Добавление нескольких разных товаров | Все выбранные товары отображаются в корзине | Not Run |
| CART-004 | Увеличение количества товара в корзине | Количество товара и итоговая стоимость увеличиваются | Not Run |
| CART-005 | Уменьшение количества товара в корзине | Количество товара и итоговая стоимость уменьшаются | Not Run |

## Проверки безопасности

| ID | Проверка | Ожидаемый результат | Статус |
|---|---|---|---|
| SEC-001 | Авторизации с SQL-инъекцией в поле Email | Вход не выполняется, отображается сообщение об ошибке | Not Run |
| SEC-002 | Выполнение XSS через поле поиска | Введённый текст безопасно обрабатывается, JavaScript-код не выполняется | Not Run |
