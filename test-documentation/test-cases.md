# Тест-кейсы OWASP Juice Shop

**ID:** TC-001

**Summary:** Авторизация с валидными данными

**Preconditions:**

- Пользователь зарегистрирован.
- Пользователь не авторизован.
- Открыта страница авторизации.

**Test Data:**

- Email: `test@mail.com`
- Password: `12345`

**Steps:**

1. Ввести зарегистрированный email в поле Email.
2. Ввести корректный пароль в поле Password.
3. Нажать кнопку "Log in".

**Expected Result:**

Пользователь успешно авторизован. Открывается главная страница приложения, в меню отображается учётная запись пользователя.

## TC-002

**ID:** TC-002

**Summary:** Авторизация с неверным паролем

**Preconditions:**

- Пользователь зарегистрирован.
- Пользователь не авторизован.
- Открыта страница авторизации.

**Test Data:**

- Email: `test@mail.com`
- Password: `1234567`

**Steps:**

1. Ввести зарегистрированный email в поле Email.
2. Ввести неверный пароль в поле Password.
3. Нажать кнопку «Log in».

**Expected Result:**

Авторизация не выполняется. Отображается сообщение «Invalid email or password.».

**Postconditions:**

Пользователь остаётся неавторизованным.

## TC-003

**ID:** TC-003

**Summary:** Добавление товара в корзину

**Preconditions:**

- Пользователь авторизован.
- Корзина пользователя пуста.
- Открыта главная страница с каталогом товаров.

**Test Data:**

- Любой доступный товар из каталога.
- Количество: 1.

**Steps:**

1. Выбрать товар в каталоге.
2. Нажать кнопку добавления товара в корзину.
3. Открыть корзину.

**Expected Result:**

Выбранный товар отображается в корзине. Счётчик корзины показывает `1`. Стоимость товара соответствует цене в каталоге.
